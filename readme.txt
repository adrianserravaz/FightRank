FightRank is a WordPress plugin for UFC fight cataloguing, automatic scoring, user ratings and custom frontend templates.

== Description ==

FightRank registers custom post types for fights, fighters and UFC events, adds custom metadata and taxonomies, calculates an automatic fight score and allows logged-in users to rate fights through an AJAX interface.

The plugin also includes an admin importer for public UFC CSV data and custom templates for archives, profiles, event pages and fight detail pages.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/fightrank-plugin`.
2. Activate the plugin through the WordPress admin panel.
3. Use `Tools > Importar UFC` to import public UFC data.

== Features ==

* Custom post types and taxonomies.
* Automatic fight scoring.
* User ratings stored in a custom table.
* AJAX rating flow with nonce validation.
* CSV importer for public UFC data.
* Custom responsive frontend templates.
